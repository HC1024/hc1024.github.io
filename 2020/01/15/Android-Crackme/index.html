三.注册码

先打开程序运行一下，要求是输入一个用户名以及此用户名对应的注册码。

工具分析

这是两段关键代码，分析即可明白，注册码即为输入的用户名取md5后去掉奇数位字符所剩下的字符串。

py脚本

脚本代码：

import hashlib

your_str = input("Please input your username: ")

m = hashlib.md5()

b = your_str.encode(encoding='utf-8')
m.update(b)
gencode = m.hexdigest()

code_list = []
i = 0
while i < len(gencode) :
code_list.append(gencode[i])
i += 2

code = ''.join(code_list)
print("Your code: " + code)
